Sales Dashboard Project

Sales dashboard project which reports top customers, current year sales and past year sales. Data cleaning was done using Power Query. Measures were created to visualise analysis and appropriate graphs were utilized to create the dashboard.

HR Analytics Dashboard

This project focused on developing an interactive HR analytics dashboard using Power BI to provide insights into workforce composition, tenure, and promotion readiness. Key metrics and features included: Gender Distribution: Analysis of male and female employee counts to support diversity and inclusion tracking. Total Headcount: A real-time view of the total number of employees across departments and roles. Service Year Breakdown: Visualization of employees grouped by years of service to monitor workforce experience levels and retention trends. Promotion Readiness: Comparative analysis of employees due for promotion versus those not yet due, helping HR teams plan career progression and talent development. Interactive Filters & Visuals: Dynamic charts, slicers, and KPIs enabled users to drill down by department, job grade, and location for deeper insight. The dashboard provided HR managers and decision-makers with a clear, data-driven view of the organization’s talent landscape, enabling proactive workforce planning and strategy.

Excel Project

Sales Performance Dashboard (Excel) This project involved designing an interactive sales dashboard using Excel to help track and analyze sales performance across regions, products, and time periods. Key features included: Pivot Tables for dynamic data summarization and quick filtering by product category, region, and sales rep. Power Pivot was used to manage large datasets efficiently, build data models, and create relationships between multiple tables (e.g., sales, targets, and customer data). Charts & Graphs (bar charts, line graphs, and slicers) were incorporated to visualize trends, compare performance against targets, and highlight top-selling products. The final dashboard allowed users to gain quick insights into overall performance, identify sales trends, and drill down into specific areas for deeper analysis. This project showcased how Excel’s advanced tools can turn raw data into an insightful, easy-to-use reporting tool for decision-makers.
